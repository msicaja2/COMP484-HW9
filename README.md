## Reflection
The easiest part was doing the number operations, such as adding and dividing. The hardest part was working on getting the parts of the date (such as getMonth), since I did not previously know how to get them. I learned that you can extract certain parts of the date from the Date object. I learned about the method isInteger for the Number object, since I did not previously know that it was a method. I learned about the actual process of displaying the results without using HTML syntax by using the document.getElementById(ID Value).textContent, which makes it much simpler to dynamically change the text using JavaScript.
